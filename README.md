VitaCare - Online Medical Report Appointment System

VitaCare is a web-based solution designed to streamline the process of obtaining medical reports from government hospitals. By replacing traditional, time-consuming physical queues with a secure and convenient online scheduling system, VitaCare ensures that patients can manage their health documentation efficiently and at their convenience.

🚀 Features
📅 Effortless Scheduling
Real-time Availability: View and select available time slots for various hospitals.

Smart Filtering: Select specific hospitals (e.g., NHSL Colombo, Castle Street Hospital) and the required report type (FBC, Thyroid Function Test, X-Ray, etc.).

💳 Integrated Payments & Discounts
Secure Checkout: Integrated payment gateway for Visa and Mastercard.

Occupational Discounts: Automated 50% discount logic specifically for University Students.

Instant Confirmation: Automated generation of a unique Token Number upon successful payment.

🌓 User Experience
Dark & Light Mode: A seamless toggle to switch between themes for better accessibility.

My Account Dashboard: A comprehensive history for users to track, view, and print their previous medical report requests and payment receipts, alongside personal profile management.

Responsive Design: Fully optimized for different screen sizes, from desktops to mobile devices.
